# Pop Drop

Pop Drop is a MERN-based location sharing web app where users can drop short messages at specific map locations and explore nearby drops in real time.

### Features
- Interactive map-based drops
- Nearby location discovery
- Like system for drops
- Splash screen & smooth UI
- Fully deployed (Vercel + Render)

### Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB (Geospatial queries)
- Deployment: Vercel (frontend), Render (backend)
